CURSOSIA - WEB PREMIUM COMERCIAL PARA NETLIFY

ESLOGAN:
"El comparador argentino de cursos de IA"

INSTRUCCIONES DE SUBIDA A NETLIFY:
1. Descomprimir este ZIP.
2. Entrar a Netlify.
3. Add new site > Deploy manually.
4. Arrastrar la carpeta completa o el ZIP.
5. Configurar el dominio cursosia.ar.
6. Redirigir cursosia.com.ar hacia cursosia.ar.

CARGA DE CURSOS CON IMÁGENES:
Opción A:
- Subir/copiá imágenes dentro de /assets/courses/.
- Editar /data/cursos.json.
- En cada curso, usar el campo "image": "nombre-de-la-imagen.jpg".
- También acepta URLs externas en "image".

Opción B:
- Abrir /admin/.
- Cargar cursos desde el formulario.
- En "Imagen de tarjeta", escribir el nombre de archivo ubicado en /assets/courses/ o pegar una URL.
- Descargar cursos.json.
- Reemplazar /data/cursos.json.
- Volver a subir a Netlify.

CURSOSIA CON PANEL ADMINISTRADOR REAL

Esta versión incorpora Decap CMS en /admin/.

PASOS:
1. Crear cuenta gratis en GitHub.
2. Crear repositorio nuevo y subir esta carpeta.
3. En Netlify: Add new site > Import from Git.
4. Elegir GitHub y el repositorio.
5. Deploy.
6. En Netlify: Identity > Enable Identity.
7. En Identity > Services activar Git Gateway.
8. En Identity > Invite users, invitar tu email.
9. Abrir https://cursosia.ar/admin/

El panel permite editar:
- SEO principal.
- Frases de portada.
- Eslogan.
- Textos del buscador.
- Métricas.
- Cursos.
- Imágenes.
- Precios.
- Links de afiliado.
- Blog SEO.
- Footer.

IMPORTANTE:
Si subís la web arrastrando ZIP manualmente, la web funciona, pero el panel no guarda cambios. Para panel real necesitás GitHub + Netlify conectado.
